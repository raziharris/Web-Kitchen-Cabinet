Website Function Brochure

A simple guide for clients and team members to understand the Cabinet Studio product selection, model browsing, and booking enquiry flow.

1. Select Series

Choose Kitchen Cabinet, Wardrobe, TV Cabinet, or Living Cabinet.

2. Browse Models

Filter by layout, material, and budget before choosing a model.

3. Configure

Select material, finishing, color, length, and site visit date.

4. Enquire

Copy a WhatsApp-ready booking message for follow-up.

Main Page

The main page introduces the product series. Users start here by choosing the cabinet category that matches their project.

Product series grid with navigation into each configuration path.
Main functions
  • Displays available product categories.
  • Shows short descriptions, typical sizes, material options, and starting price.
  • Each product card links to the model overview page.
Client benefit
  • Easy first-step selection.
  • No payment required online.
  • Clear path toward booking a site visit.

Product Overview Page

After a product series is selected, the user sees model options with filters and technical details.

Model browsing page with filters, search, model cards, and configure actions.
Main functions
  • Search available models.
  • Filter by layout, material, and budget.
  • View model starting price, size, lead time, and storage details.
User action
  • Click Technical Data for more detail.
  • Click Configure This Model to continue.
  • Return to product series selection using the back button.

Configure Page

This page is where users personalize the selected model before creating a booking enquiry for a technical site visit.

Full-page configurator for material, finishing, color, size, date, and notes.
Main functions
  • Select material and finishing.
  • Choose color swatches.
  • Enter cabinet length and preferred site visit date.
  • Write client notes.
Output
  • Shows estimated starting price.
  • Creates a WhatsApp-ready enquiry message.
  • Final quotation and payment happen offline after site measurement.
